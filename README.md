# Fake News Detection — AI

Detects whether a news article is **FAKE** or **REAL** using NLP and machine learning.

---

## Architecture

```
Frontend (React + Vite)  :4000
        ↓
Backend (Node.js + TypeScript)  :3000
        ↓
ML Service (Python + FastAPI)  :8000
        ↓
TF-IDF + Logistic Regression model
```

---

## ML Approach

1. Text preprocessing — lowercasing, punctuation removal, stopword filtering
2. Feature extraction — TF-IDF vectorization (5000 features)
3. Classification — Logistic Regression
4. Evaluation — accuracy, precision, recall, F1, confusion matrix

**Accuracy: ~90–95%**

---

## Dataset

Kaggle - [Fake and Real News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset) by Clément Bisaillon

Download `Fake.csv` and `True.csv` and place them in `data/`. This folder is gitignored _(each contributor must download the dataset locally)_.

---

## Getting Started

### Prerequisites

- Python 3.10+
- Node.js 22+

### ML Service

```bash
# from project root
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

mkdir data      # add Fake.csv and True.csv here
mkdir outputs

python -m ml.train.main                        # train and save model
uvicorn ml.service.main:app --port 8000        # start ML service
```
