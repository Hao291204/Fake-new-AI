# Fake News Detection — AI

Detects whether a news article is **FAKE** or **REAL** using NLP and machine learning.

---

## Dataset

Kaggle - [Fake and Real News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset) by Clément Bisaillon

Download `Fake.csv` and `True.csv` and place them in `data/`. This folder is gitignored _(each contributor must download the dataset locally)_.

---

## Getting Started

### Prerequisites

- Python 3.10+
- Node.js 22+

### ML Service

```bash
# from project root
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

mkdir data # add Fake.csv and True.csv here
mkdir outputs

python -m ml.train.main # train and save model
uvicorn ml.service.main:app --port 8000 # start ML service
```
