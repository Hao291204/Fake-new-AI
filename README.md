# Fake News Detection — AI

Detects whether a news article is **FAKE** or **REAL** using NLP and machine learning.

---

## Dataset

Kaggle - [Fake and Real News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset) by Clément Bisaillon

Download `Fake.csv` and `True.csv` and place them in `data/`. This folder is gitignored _(each contributor must download the dataset locally)_.

---

## Getting Started

### Prerequisites

- Python 3.10+
- Node.js 22+

### First time setup

```bash
# from project root
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

mkdir data # add Fake.csv and True.csv here
cd frontend && npm install && cd ..
```

### Running the app

```bash
source .venv/bin/activate
./start.sh
```

The start script will train the model automatically if it hasn't been trained yet, then start both the ML service and the frontend.

- Frontend: http://localhost:5173
- ML service: http://localhost:8000
